# IA_em_Sala_de_Aula
## Entregas para a conclusão do curso: Bootcamp Santander EducaIA
1. Descrição do projeto de conclusão:
  - Criação de um plano de aula interativo e personalizado utilizando ferramentas do econssistema IA. A utilização das ferramentas deverão contemplar:
    -  Princípios de ética
    -  Proteção de dados
    -  Personalização
    -  Otimização do tempo

2. Detalhamento das etapas (Disciplina: Introduçao à Ciência de Dados para alunos do cursos superior em administração.)
  - Utilização do ChatGPT para gerar o esqueleto do plano de aula, incluindo objetivos de aprendizagem, conteúdo programático e metodologia.)
  - Desenvolver materiais complementares (textos, exercícios, exemplos) com o Microsoft Copilot no Word e/ou ChatGPT, garantindo a agilidade na produção.
  - Griar uma apresentação visualmente impactante para a aula utilizando o Microsoft Copilot no PowerPoint, com slides que resumam o conteúdo e engajem os alunos.
  - Internacionalizar um trecho do material, realizando revisão inteligente e traduçao rápida (Word + IA) para considerar a inclusão de alunos com outras linguas.
  - Treinar o ChatGPT para adaptar parte do material ou criar atividades adicionais para, no mínimo, dois perfis de alunos diferentes (ex.: nenhum conhecimento digital e iniciante digital.)
  - Criat rubricas de avaliação
  - As entregas deverão ser feitas no Github

3. Entregas:
  - Plano de aula detalhado em Word ou PDF
    - Nome do arquivo: Plano_de_Aula_IA.pdf.
  - Plano de aula traduzido para o Inglês
    - Nome do arquivo: Plano_de_Aula_IA_Ingles.pdf
  - Apresentção em PowerPoint
    - Nome do arquivo: Apresentacao_Aula_IA.pptx
  - Documento de refelxão
    - Nome do arquivo: Refelxão_IA.doc
  - README.md (não esquecer de colocar os créditos e autoria)
  - prompts.md (incluindo todos os prompts utilizados)
